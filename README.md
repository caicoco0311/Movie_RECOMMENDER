# 🤖 Cai's AI Chatbot 

这是一个基于 **Streamlit** 和 **OpenAI API** 开发的轻量级聊天机器人工作站。它专为创作者和技术研究者设计，支持多轮对话记忆、参数调节以及角色切换。

---

## ✨ 核心功能

- **多轮对话记忆**：支持上下文关联，能够进行连续的逻辑推导。
- **流式回复 (Streaming)**：如同打字机般的实时响应体验。
- **动态参数调节**：在侧边栏实时调整 `Temperature`（创造力）和模型版本。
- **部署友好**：针对 Streamlit Cloud 进行了优化，一键部署。

## 🛠️ 技术栈

- **Frontend**: [Streamlit](https://streamlit.io/)
- **LLM Engine**: [OpenAI API](https://platform.openai.com/) (GPT-4o / GPT-4 Turbo)
- **Language**: Python 3.9+
